# Marnov Stability Criterion

## Definition

Let:

C(t) ≥ 0 — coherence  
P(t) ≥ 0 — dissipative load  

## Necessary Condition

A system remains in a structurally stable regime if:

C(t) ≥ P(t) + ε,  ε > 0

## Interpretation

- C > P → stable regime  
- C = P → critical regime  
- C < P → unstable regime  

## Extended Form

dC/dt = αC − βP − γ(Δφ)²

## Notes

This condition is necessary but not sufficient for full phase coherence.
