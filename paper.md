2. CORE VARIABLES

S(t) — synthesis (structure formation)
P(t) — load / extraction
D(t) — dissipation (losses)
C(t) — coherence (internal alignment)

---

3. FUNDAMENTAL EQUATION

dX/dt = S(t) − P(t) − D(t)

Define:

Δ(t) = S(t) − P(t) − D(t)

---

4. STABILITY CONDITION

Local condition:

Δ(t) ≥ ε > 0

Integral condition:

∫(S − P − D) dt > 0

---

5. COHERENCE LINK

C(t) modulates system efficiency:

S_eff = C · S
D_eff = D / C

---

6. DYNAMIC EQUILIBRIUM

S(t) ≈ P(t) + D(t)

System maintains structure under continuous flow.
